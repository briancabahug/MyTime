# Ticket 4: Create Storage Service
Priority: HIGH
Estimated Time: 30 minutes

## Description
Implement the storage service layer that handles all Hive database operations.

## Tasks

  - Create folder lib/services/
  - Create lib/services/storage_service.dart
  - Implement init() method with:
    - Hive initialization
    - All adapter registrations
    - Box opening for each data type

  - Implement timeline entry methods:
    - saveEntry()
    - getEntriesForDay()
    - deleteEntry()
    - updateEntry()

  - Implement alarm methods:
    - saveAlarm()
    - getAllAlarms()
    - deleteAlarm()

  - Implement alarm log methods:
    - saveAlarmLog()
    - getAlarmLogsForDay()

  - Implement summary methods:
    - saveDailySummary() / getDailySummary()
    - saveWeeklySummary() / getWeeklySummary()

  - Add helper method _dateKey() for consistent date-based keys

## Acceptance Criteria

  - StorageService class is complete with all methods
  - All methods have proper async/await patterns
  - Date filtering works correctly for day-based queries
  - Box names are consistent across methods

## Notes
The service uses static methods for simplicity in MVP. We can refactor to singleton pattern later if needed.
